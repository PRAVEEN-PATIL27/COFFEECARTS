### ☕ Coffee Shop Ordering System

The **Coffee Shop Ordering System** is a beginner-friendly Python console application that simulates a basic coffee shop ordering process. The project allows users to view a menu, select coffee items, add them to a shopping cart, and generate a final bill at checkout. It demonstrates the practical use of Python fundamentals such as dictionaries, lists, functions, loops, conditional statements, and user input handling.

The application maintains a predefined menu containing different coffee varieties and their prices. Customers can continuously add items to their cart until they choose to complete their order. Once checkout is initiated, the program calculates the total cost, displays an itemized bill, and generates the final payment amount. The system also handles invalid menu selections and prevents checkout when the cart is empty.

This project serves as an excellent introduction to building command-line applications and understanding the core concepts of Python programming. It can be further enhanced by adding features such as quantity selection, order removal, discounts, payment processing, inventory management, and graphical user interfaces.

### Key Features

* ☕ Display available coffee menu items and prices
* 🛒 Add selected items to a shopping cart
* ✅ Validate customer selections
* 💰 Calculate total bill automatically
* 📋 Generate itemized order summary
* 🔄 Continuous ordering until checkout
* 🚫 Empty cart validation
* 🐍 Built entirely using Python
